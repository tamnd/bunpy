# v0.8.1 ‚Äî bunpy.node.path: Node.js path shim

Adds `bunpy.node.path` ‚Äî a Node.js `path` module shim backed by `path/filepath`.

## New

- `path.join(*parts)` ‚Äî join path segments
- `path.resolve(*parts)` ‚Äî join + make absolute
- `path.dirname(p)` ‚Äî directory component
- `path.basename(p, ext?)` ‚Äî filename, optional ext strip
- `path.extname(p)` ‚Äî extension including dot
- `path.relative(from, to)` ‚Äî relative path between two paths
- `path.isAbsolute(p)` ‚Äî bool
- `path.normalize(p)` ‚Äî clean up `..` and `.` segments
- `path.sep` ‚Äî OS path separator constant
- `path.delimiter` ‚Äî OS path-list delimiter (`:` or `;`)

## Usage

```python
import bunpy.node.path as path
full = path.join("/usr", "local", "bin")
ext = path.extname("script.py")   # ".py"
```
